# Coin API Backend

## Hızlı Başlangıç

### 1. Paketleri Yükle
```bash
npm install
```

### 2. .env Dosyası Oluştur
`.env.example` dosyasını kopyalayıp `.env` yapın ve değerleri düzenleyin.

### 3. Sunucuyu Başlat
```bash
npm run dev
```

Sunucu `http://localhost:3000` adresinde çalışacak.

## Endpoints

- `POST /api/auth/register` - Kayıt
- `POST /api/auth/login` - Giriş
- `POST /api/auth/logout` - Çıkış

## Test

Tarayıcıda `http://localhost:3000` adresine gidin. "Coin API Server Çalışıyor!" mesajını görmelisiniz.
